## Цель работы

Разработать мобильное приложение на Android (Kotlin) для взаимодействия с Spring Boot, обеспечивающее управление через мобильные устройства.

## Выполнение работы

Создан API клиент с Retrofit:

1. Настроен базовый URL для эмулятора (http://10.0.2.2:8080/)

2. Реализованы все необходимые endpoints

3. Добавлен логгинг запросов для отладки

Разработан пользовательский интерфейс:

1. MainActivity - главный экран с навигацией

2. MembersActivity - список участников

3. MemberDetailActivity - детальная информация об участнике

4. WorkoutsActivity - список тренировок

5. Адаптивная верстка с ConstraintLayout

Реализована бизнес-логика:

1. Асинхронные запросы через Kotlin Coroutines

2. Обработка ошибок сети

3. Кэширование данных при необходимости

4. Обновление UI в главном потоке

Добавлены дополнительные функции:

1. Pull-to-Refresh для обновления списков

2. Поиск и фильтрация участников

3. Валидация данных при добавлении

4. Поддержка темного режима

Настроена сборка и развертывание:

1. Конфигурация Gradle для разных сборок

2. Минимальная версия API 24 (Android 7.0)

3. Разрешения для доступа в интернет

## Выводы

Разработано мобильное приложение для Android, полностью интегрированное с Spring Boot сервером

Приложение использует современные технологии Android разработки (Kotlin, Coroutines, Retrofit)

Реализован удобный и отзывчивый пользовательский интерфейс

Обеспечена стабильная работа при плохом соединении с сетью
